Question-1: What are the different ways to select an element in the DOM?
Answer:There are several ways to select an element in the DOM: select one element by its id. Select all elements with the same class. Select all elements with the same tag.

Question-2: What is the difference between innerHTML, innerText, and textContent ?
Answer: HTML content (including tags). visible text (ignores hidden text). all the text (even hidden text).

Question-3: What is event delegation in the DOM?
Answer:Event delegation is a technique where you add a single event listener to a parent element instead of adding individual listeners to each child element.

Question-4: What is event bubbling in the DOM?
Answer:Event bubbling is when an event starts on a child element and then moves up to its parent elements one by one.

Question-5: How do you create, add, and remove elements using JavaScript?
Answer:To create, add, and remove elements using JavaScript, you use document.createElement() to create, appendChild() to add, and removeChild() to remove the element.
